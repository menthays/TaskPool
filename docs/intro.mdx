# Introduction

_TaskPool_ manage timer tasks

## Install

### With npm

```sh
npm i @menthays/taskpool
```

## Quick Examples

```js
import TaskPool from "@menthays/taskpool";
```

```js
const pool = new TaskPool();
pool.newTask("goto", { dest: "disney" }, console.log, 5000);
pool.finishTask("goto", _ => console.log('Execute ahead of time'));
// {"dest": "disney"}
// Execute ahead of time
```

or 

```js
const pool = new TaskPool();
pool.newTask("goto", { dest: "disney" }, console.log, 5000);
pool.abortTask("goto", _ => console.log('Abort Task'));
// Abort Task
```
